## ⚙️ Requisitos

- Python 3.8+
- Instalar as dependências:

```bash
pip install -r requirements.txt
```
## 🎁 Iniciando

Agora basta apenas inicilizar o CMD em sua pasta (a que o arquivo foi salvo).
E utilizar:

```bash
py main.py
```
